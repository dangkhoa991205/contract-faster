Design — Phần 3: AI Features & API

Cách AI hoạt động và các API endpoints

3 tính năng AI và cách hoạt động

🔍 1. Placeholder Detection (khi upload template)
Input: file .docx
↓ mammoth.js đọc text
↓ Gửi lên OpenAI GPT-4o
↓ Prompt: "Tìm tất cả placeholder
  trong hợp đồng này (VD: {{TEN_BEN_A}},
  [Ngày ký], ...)"
Output: JSON list các field cần điền
Kết quả trả về:
[{name: "TEN_BEN_A", label: "Tên bên A", type: "text"},
 {name: "NGAY_KY", label: "Ngày ký", type: "date"},
 {name: "GIA_TRI", label: "Giá trị HĐ", type: "number"}]
✨ 2. Smart Fill (khi điền form)
Input: {field: "GIA_TRI", context: "HĐ dịch vụ thiết kế web"}
↓ Gửi lên OpenAI
↓ Prompt: "Gợi ý giá trị hợp lý
  cho trường này dựa vào context"
Output: suggestion + explanation
Hiển thị:
💡 "Thông thường 20-50 triệu VND cho dự án thiết kế web cơ bản"

Cũng nhớ giá trị từ hợp đồng trước (từ DB) để auto-suggest
💬 3. Contract Chat (sidebar AI)
Input: {question: "Điều khoản phạt nên bao nhiêu?",
  contractContent: "..."}
↓ OpenAI với system prompt
  "Bạn là chuyên gia pháp lý VN..."
Output: câu trả lời + disclaimer
Lưu ý:
- Lưu lịch sử chat vào bảng AiChat
- Luôn kèm disclaimer "Đây không phải tư vấn pháp lý chính thức"
- Rate limit theo plan (Solo: 20 msg/ngày)

API Routes (Next.js App Router)

POST /api/auth/[...nextauth] → NextAuth Google OAuth
POST /api/templates → Upload + detect placeholders
GET /api/templates → Lấy danh sách templates
POST /api/contracts → Tạo hợp đồng mới
GET /api/contracts → Lấy danh sách hợp đồng
POST /api/contracts/[id]/export → Xuất PDF hoặc DOCX
POST /api/ai/detect-placeholders → GPT-4o nhận diện fields
POST /api/ai/smart-fill → Gợi ý giá trị cho field
POST /api/ai/chat → Chat về hợp đồng
POST /api/stripe/checkout → Tạo Stripe checkout session
POST /api/stripe/webhook → Xử lý Stripe events
GET /api/user/usage → Kiểm tra quota còn lại

Document Generation Flow

User submit form → Load template .docx từ storage → docxtemplater điền placeholder → Xuất DOCX
Template lưu trữ: giai đoạn 1 dùng local /uploads → giai đoạn 2 migrate sang AWS S3
PDF: convert từ DOCX bằng thư viện (libreoffice hoặc API)
