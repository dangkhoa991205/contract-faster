Template-First SaaS · Next.js 16 · Prisma · Stripe · OpenAI

Luồng chính (Core Flow)

User Journey
Đăng nhập (Google) → Chọn / Upload Template → AI nhận diện placeholder → Điền form (AI gợi ý) → Preview hợp đồng → Xuất PDF / DOCX
+ AI Chat sidebar: giải thích điều khoản, gợi ý nội dung bất kỳ lúc nào

App (yêu cầu đăng nhập)
  • Dashboard (/app)
  • Template library (/app/templates)
  • Contract editor (/app/contracts/new)
  • Contract list (/app/contracts)
  • Settings + Billing (/app/settings)
